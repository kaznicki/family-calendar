    Implementation requirements:
    1. Render a fixed `z-50` div positioned at anchorPos.x/y (top/left) with white bg, border, rounded corners, shadow — matching the style of the existing EventPopover (bg-white, border border-gray-200, rounded, shadow-lg, py-1, text-sm).
    2. Single button that shows "Mark as holiday" when !isHoliday, "Unmark holiday" when isHoliday. On click: call the appropriate handler then call onClose().
    3. Add an invisible backdrop div (fixed inset-0 z-40, no bg) that calls onClose on click — this handles "click outside to close". The menu itself must be z-50 (above the backdrop at z-40).
    4. The menu button should have `block w-full text-left px-4 py-1.5 hover:bg-gray-50` classes.

    Example structure:
    ```tsx
    export function HolidayMenu({ isHoliday, onMark, onUnmark, onClose, anchorPos }: HolidayMenuProps) {
      return (
        <>
          <div className="fixed inset-0 z-40" onClick={onClose} />
          <div className="fixed z-50 bg-white border border-gray-200 rounded shadow-lg py-1 text-sm"
               style={{ top: anchorPos.y, left: anchorPos.x }}>
            <button
              type="button"
              className="block w-full text-left px-4 py-1.5 hover:bg-gray-50"
              onClick={() => { isHoliday ? onUnmark() : onMark(); onClose() }}
            >
              {isHoliday ? 'Unmark holiday' : 'Mark as holiday'}
            </button>
          </div>
        </>
      )
    }
    ```
